# Lok Sabha Election Data Analysis & Visualization

This project analyzes Indian Lok Sabha election candidate data from 2004, 2009, 2014, and 2019 using Python.  
It applies descriptive statistical measures and visualizations to study candidate demographics, education, assets, liabilities, and criminal records, providing insights into electoral transparency and trends.

## Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  

## Features
- Descriptive statistical analysis  
- Data visualization (histograms, bar charts, pie charts)  
- Multi-year election data comparison  

## Dataset
Lok Sabha candidate data sourced from publicly available election records.
